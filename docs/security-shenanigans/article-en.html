How to Configure a Home SIEM

Wazuh is an open-source SIEM and XDR platform for centralized security monitoring across endpoints and infrastructure.

It enables log collection, correlation, and incident detection in real time.

1. Preparing the Virtual Machine

- 4 GB RAM
- 2 CPU cores
- 20–40 GB disk
- stable network connection

Resource usage increases with number of agents.

2. Installing the Wazuh Server

curl -sO https://packages.wazuh.com/4.7/wazuh-install.sh
sudo bash wazuh-install.sh -a

3. Adding Agents

Agents forward system and security logs to the Wazuh manager.

4. Firewall Logs

/var/ossec/logs/archives/archives.log
sudo systemctl restart wazuh-manager

5. Decoder Setup

Used to structure raw log data into fields.

6. Rules

<rule id="100040" level="3">
  <decoded_as>Firewall_Generic</decoded_as>
  <description>Firewall Log Event</description>
</rule>

7. Testing

/var/ossec/bin/wazuh-logtest

Phase 2 = decoding, Phase 3 = alert generation.
