Kako konfigurirati kućni SIEM

Wazuh je open-source SIEM i XDR platforma za centralizirano prikupljanje i analizu sigurnosnih događaja.

Koristi se za monitoring logova, detekciju i korelaciju incidenata.

1. Priprema VM-a

  • 4 GB RAM
  • 2 CPU
  • 20–40 GB disk
  • Mrežna povezanost

2. Instalacija Wazuh servera

curl -sO https://packages.wazuh.com/4.7/wazuh-install.sh
sudo bash wazuh-install.sh -a

3. Agenti

Agenti šalju sistemske i sigurnosne logove na server.

4. Firewall logovi

/var/ossec/logs/archives/archives.log
sudo systemctl restart wazuh-manager

5. Decoder

Koristi se za parsiranje logova u strukturu.

6. Pravila

<rule id="100040" level="3">
  <decoded_as>Firewall_Generic</decoded_as>
  <description>Firewall event</description>
</rule>

7. Testiranje

/var/ossec/bin/wazuh-logtest
